# Esports Team Management and Tournament Operations Discord Bot

A Discord bot for esports organizations, gaming tournament organizers, and competitive gaming communities. Manages team rosters with game-specific role assignments, runs single and double elimination bracket tournaments with automated result recording, posts live match score updates, tracks individual player statistics across sessions, sends scrim scheduling notifications between teams, and delivers weekly team performance digests with win rate and KDA trend analysis.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
